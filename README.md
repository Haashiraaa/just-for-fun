A small collection of playful experiments, scripts and little projects — a personal playground for learning and trying out ideas.

What you'll find
- Small scripts and experiments
- One-off projects and prototypes
- Miscellaneous utilities and demos

Quick start
1. Clone the repo:
   git clone https://github.com/Haashiraaa/just-for-fun.git
2. Explore the files in the repository root and subfolders.

Contributing
- This repo is casual — feel free to open issues or PRs with improvements, fixes, or new fun toys.
- Keep changes small and self-contained; include a short description of what the addition/demo does.
